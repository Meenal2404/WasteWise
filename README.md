# WasteWise — Municipal Waste Management Analysis

## Problem Statement
A municipal ward was facing unmanaged waste collection with no data-driven insights. 
This project analyzes 6 months of waste data to identify problem areas and recommend 
a lifetime solution for efficient waste management.

## Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- MySQL 8.0
- Power BI Desktop
- VS Code

## Key Findings
- Govindpura generates highest waste: 50,228 kg
- 51.1% waste collection is MISSED — critical problem identified
- January has peak waste generation: 51,292 kg
- Govindpura has highest complaints average: 5.40 per day

## Recommendations
1. Deploy extra collection vehicles in Govindpura and Bairagarh
2. Introduce dry/wet waste segregation bins in all 6 areas
3. Increase collection frequency in January and March
4. Launch awareness campaign in high-complaint zones

## Project Structure
- data/raw — Original generated data
- data/cleaned — Cleaned dataset
- notebooks — Python analysis files
- sql — SQL analysis queries
- dashboard — Power BI dashboard
- report — Final recommendations

## Dataset
Synthetic dataset of 1,080 records across 6 areas, 6 months (Jan-Jun 2025)
